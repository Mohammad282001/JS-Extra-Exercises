## Q1: Evaluate JavaScript Expressions

1. `-5*3`  
   **Value:** -15

2. `"JavaScript " + 50`  
   **Value:** "JavaScript 50"

3. `17 % 5`  
   **Value:** 2

4. `5 % 17`  
   **Value:** 5

5. `5/10`  
   **Value:** 0.5

6. `(4 === '4')`  
   **Value:** false

7. `(4 != 5)`  
   **Value:** true

8. `(7 <= 8)`  
   **Value:** true

9. `"Hello" + 5`  
   **Value:** "Hello5"

10. `Math.ceil(x) - Math.floor(x)`  
    **Value:** The value of `x` needs to be provided to evaluate.

11. `Math.pow(x,2)`  
    **Value:** The value of `x` needs to be provided to evaluate.
